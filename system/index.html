The System

One chair. Every configuration.

The Chariot is the center of everything. It ships as a high-spec manual chair. Every ParaBoards attachment connects to it. You start with what you need and add from there — all using parts you can source yourself.

How the system connects

You don't need everything to start

The Chariot alone is a capable everyday chair. It ships with the castor attachment as standard. If you want power, you add a propulsion attachment and a Battery Box. If you want options, you can own more than one attachment and swap based on the day.

Every part that can be sourced from a common vendor is. No proprietary connectors for the sake of it. If ParaBoards LLC closed tomorrow, your rig still works — and you can still get parts.
